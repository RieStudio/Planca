# Planca

Planca is a minimalist, distraction-free daily task and todo manager focused entirely on mental clarity and simplicity. Designed to eliminate cognitive load, its clean and intuitive interface ensures users know exactly what to do the moment they open the app. Effortlessly organize your day and categories without any unnecessary noise.

Built to be fast, modern, and light—keeping you focused on what truly matters.

##  Key Features
- **Minimalist UI/UX:** Clean aesthetics, soft tones, and smooth transitions designed to prevent eye strain.
- **Effortless Navigation:** Quick-access side drawer for custom categories and time-based filters (Today & Upcoming).
- **Customizable Widgets:** Clean home screen widgets with real-time appearance and opacity customization.

##  Architecture & Tech Stack
- **Kotlin Multiplatform (KMP):** Shared business logic and data layer across Android and iOS.
- **Compose Multiplatform:** Shared, high-performance UI components ensuring an identical minimalist look on both platforms.
- **Room KMP:** Local, multiplatform SQLite database for robust offline task management.
